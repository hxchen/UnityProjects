- 刚体的三种类型

  Dynamic	普通刚体，有质量

  Static	静态刚体，质量无穷大（适用于建筑物、地面等固定不动的物体）

  Kinematic	运动学刚体，质量为0（物理规律不起作用，一般用于碰撞检测）
